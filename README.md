<h1 align="center">Учебный проект сайта подбора велосипедов</h1>
<p align="center">выполнен в рамках 4 месяца обучения</p>
<h3>Краткое описание:</h3>
<p>Проект представляет из себя сайт, распологающийся на одной странице. Сайт рассказывает какие бывают типы поверхностей для передвижения на велосипеде, чем они отличаются и какие модели велосипедов подходят для каждого типа поверхности.Для написания использовались языки: HTML, CSS и JS.</p>
